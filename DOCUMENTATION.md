# Godot HitflashAnimation API Reference
Generated: 2026-03-09

A godot addon that can be used to apply hitlfash animations to nodes

### 🐧 Linux / MacOS
To add this to your project, copy paste this command into terminal at the root of your project:
```bash
clone-gd-addon https://github.com/ChillCube/Godot_HitflashAnimation.git
```

## Class: HitFlash

This node is attached to a sprite or animated sprite to create a hitlflash effect. This can be used for things like indicating damage.
