# Godot_HitflashAnimation

A godot addon that can be used to apply hitlfash animations to nodes

## 📥 Installation
### Linux / MacOS
Copy and paste these commands into terminal at the root of your project:
```bash
clone-gd-addon https://github.com/ChillCube/Godot_HitflashAnimation.git
```

Full documentation can be found in [DOCUMENTATION.md](./DOCUMENTATION.md)
